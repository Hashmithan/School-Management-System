# School Management System

## Overview
The School Management System is a **Windows Forms application** developed using **C#** with **SQL Server** for database management.  
It helps school administrators efficiently manage students, teachers, subjects, sections, and enrollments through modular forms connected to a centralized dashboard.

## Functional Modules
1. **Student Management** – registration, CRUD operations, DataGridView display  
2. **Enrollment Management** – assign students to sections/years, batch enrollment  
3. **Teacher Management** – maintain teacher records and contact info  
4. **Subject Management** – assign subjects to teachers and classes  
5. **Section Management** – create/modify sections, track schedules and capacity  
6. **Search Functionality** – search by ID or name for students, teachers, and sections  

## Technical Implementation
- **Database**: SQL Server with normalized tables (Students, Teachers, Subjects, Enrollments, Sections, Attendance)  
- **UI**: Windows Forms with modular forms, input validation, and DataGridView  
- **Reporting**: RDLC Reports integrated for report generation
